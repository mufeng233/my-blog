# 沐朽阁API管理系统

## 前言
在我没有一套完整的管理程序前，我在每次添加API时都要自己打开html文件去自己编辑才可以，而且又麻烦，就很烦，于是就有了这么一个系统。
这个系统从2020年9月开发一直到现在，也是经历了挺长时间了，也感谢那几个一直支持我的朋友！

## 功能
+ 添加，修改，删除接口
+ 添加，修改，删除友情链接
+ 管理你的站点信息
+ 访问日志
+ 使用邮件接口反馈
+ 后台登陆记录
+ 自定义添加公告
+ 接口调用统计 等等等...更多功能等你来探索！
